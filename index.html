Manual do Utilizador

Esta ferramenta foi construída especificamente para garantir que qualquer membro da equipa possa estruturar relatórios, atas, editais e comunicações formais sem ferir as diretrizes do Manual de Marca TETO.

O que o sistema faz?

  • Padronização Automática: Força o uso exclusivo das fontes institucionais (Fredoka para impacto/proximidade e Montserrat para leitura formal).
  • Biblioteca Integrada: Carregue imagens via CSV (Google Sheets), JSON ou Drive.
  • Arrastar e Soltar Textos: Reordene seus títulos e parágrafos livremente segurando no botão de arraste.
  • Arrastar e Soltar Grafismos: Movimente tramas e elementos gráficos livremente no canvas com o mouse.
  • Controle Cromático Rico: Permite escolher entre a paleta oficial (Ciano, Amarelo, Verde, Vermelho, Roxo) para títulos.
  • Exportação Inteligente: Crie PDFs, exporte o projeto em JSON ou renderize o canvas exato em arquivo PNG para evitar cortes do navegador.
